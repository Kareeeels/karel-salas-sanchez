PM (PROJECT MANAGEMENT) - Metodología PSP

Esta carpeta contiene los documentos relacionados con la GESTIÓN DE PROYECTO del Personal Software Process (PSP).

En esta fase se realizan las siguientes actividades:
- Seguimiento y control del progreso del proyecto
- Gestión de recursos y asignación de tareas
- Monitoreo de métricas y indicadores de rendimiento
- Comunicación con stakeholders
- Gestión de riesgos y problemas
- Control de cambios y versiones
- Reportes de estado y avance
- Coordinación entre fases del proyecto

La gestión de proyecto es transversal a todas las fases del PSP y asegura que el proyecto se mantenga en tiempo, presupuesto y calidad. Incluye la recolección de datos para mejorar procesos futuros.
