TEST - Metodología PSP

Esta carpeta contiene los documentos relacionados con la fase de PRUEBAS del Personal Software Process (PSP).

En esta fase se realizan las siguientes actividades:
- Diseño de casos de prueba y escenarios de testing
- Ejecución de pruebas unitarias, de integración y de sistema
- Pruebas de funcionalidad y rendimiento
- Detección, registro y seguimiento de defectos
- Validación de requerimientos y criterios de aceptación
- Pruebas de regresión
- Documentación de resultados de pruebas
- Verificación de la calidad del software

La fase de testing es fundamental para asegurar la calidad del producto final. Permite identificar y corregir errores antes de la entrega, validar que el software cumple con los requerimientos y garantizar su funcionamiento correcto.
