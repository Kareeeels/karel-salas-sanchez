PLANNING - Metodología PSP

Esta carpeta contiene los documentos relacionados con la fase de PLANIFICACIÓN del Personal Software Process (PSP).

En esta fase se realizan las siguientes actividades:
- Definición de objetivos y alcance del proyecto
- Estimación de tiempo y recursos necesarios
- Identificación de riesgos potenciales
- Creación del cronograma de trabajo
- Definición de criterios de calidad
- Planificación de las fases de desarrollo

La planificación es fundamental para el éxito del proyecto, ya que permite establecer una base sólida para el desarrollo del software y ayuda a prevenir problemas futuros mediante una preparación adecuada.
